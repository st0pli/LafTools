# note

to make this application run, you should provide below env variables.

```config
export DB_USER=
export DB_PASSWORD=
export DB_HOST=
export DB_PORT=
export DB_DATABASE=
export TYKEY=
```

define them in Github action

```
S2_DB_DATABASE
S2_DB_USER
S2_DB_PASSWORD
S2_DB_HOST
S2_DB_PORT
S2_TYKEY
```
